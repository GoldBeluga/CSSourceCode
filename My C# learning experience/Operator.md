## ++
`++` is a operator that can add `1` before or after use the varaible.

Exp :
```cs
int i = 1;
Console.WriteLine(i++);
```

Output :`1`

Explanation : It wiil print `1` because use put `++` after `i`, so the variable `i` wiil add `1` after print `i`. If there is `++i`, that is wiil print `2`
